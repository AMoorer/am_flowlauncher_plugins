# Release Checklist

Use this checklist when preparing a new release.

## Pre-Release

### Code Quality
- [ ] All features complete and tested
- [ ] Run `test.py` in plugin directory - all tests pass
- [ ] Test editor with real shortcuts
- [ ] Test bookmark import from multiple browsers
- [ ] Test on clean Windows installation (if possible)
- [ ] Code reviewed for security issues
- [ ] No hardcoded paths or credentials

### Documentation
- [ ] README.md updated with new features
- [ ] CHANGELOG.md updated with version and changes
- [ ] Version number updated in:
  - [ ] `plugin.json` (plugin version)
  - [ ] `editor.py` (About dialog)
  - [ ] README.md
  - [ ] CHANGELOG.md
- [ ] Screenshots updated (if UI changed)
- [ ] All documentation links verified

### Build
- [ ] Clean build environment
- [ ] Test PyInstaller build script:
  ```bash
  cd ShortcutsEditor
  build_exe.bat
  ```
- [ ] Test standalone .exe on different machine (no Python)
- [ ] Verify .exe file size reasonable (<50MB)
- [ ] Test .exe dark mode support

## Release Process

### 1. Version Bump
```bash
# Update version in these files:
# - Flow.Launcher.Plugin.Shortcuts/plugin.json
# - ShortcutsEditor/editor.py (About dialog)
# - README.md
# - CHANGELOG.md
```

### 2. Build Packages

#### Plugin Package
```bash
# Create plugin zip
cd Flow.Launcher.Plugin.Shortcuts
# Exclude: __pycache__, *.pyc, test files
# Include: main.py, plugin.json, requirements.txt, Images/
```

#### Standalone Editor
```bash
cd ShortcutsEditor
build_exe.bat
# Verify: dist/ShortcutsEditor.exe created
```

### 3. GitHub Release
- [ ] Create new release on GitHub
- [ ] Tag version: `v1.0.0`
- [ ] Release title: `Flow Launcher Shortcuts Plugin v1.0.0`
- [ ] Copy relevant section from CHANGELOG.md to release notes
- [ ] Upload artifacts:
  - [ ] `Flow.Launcher.Plugin.Shortcuts.zip`
  - [ ] `ShortcutsEditor.exe`
  - [ ] Source code (auto-generated by GitHub)

### 4. Post-Release
- [ ] Test installation from release downloads
- [ ] Verify plugin installs correctly in Flow Launcher
- [ ] Verify standalone .exe runs without Python
- [ ] Update any external links to point to new release
- [ ] Announce on relevant channels (if applicable)

## Common Issues

### Build Issues
- **PyInstaller not found**: `pip install pyinstaller`
- **Missing modules**: Ensure all dependencies in `requirements.txt`
- **Large .exe file**: Consider using `--exclude-module` for unused packages

### Plugin Issues
- **Plugin not loading**: Check Python version matches Flow Launcher
- **Module not found**: Install dependencies in correct Python environment
- **Icons not showing**: Verify icon paths are relative to plugin directory

### Editor Issues
- **Qt platform plugin error**: Install PySide6 correctly
- **Bookmark import fails**: Browser might have bookmark file locked
- **Dark mode not working**: System theme not detected

## Version Numbers

Format: `MAJOR.MINOR.PATCH`

- **MAJOR**: Incompatible API changes
- **MINOR**: New features, backward compatible
- **PATCH**: Bug fixes, backward compatible
